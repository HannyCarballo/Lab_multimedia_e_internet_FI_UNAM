El internet 1.1. ¿Qué es el internet? El internet es una red mundial en donde hay redes locales conectadas entre sí. Se reune información, personas, computadoras y software. Nació con fines militares y no con fines comerciales, la primera conexión entre redes fue APARNET y nación con el auge de la guerra fría. El término internet es diferente al de web. Web también es conocido como World Wide Web o red informática mundial. 1.2. ¿Qué es HTTP? El http (del inglés HyperText Transfer Protocol o Protocolo de Transferencia de Hiper Textos) es el protocolo de transmisión de información de la World Wide Web, es decir, el código que se establece para que el computador solicitante y el que contiene la información solicitada puedan “hablar” un mismo idioma a la hora de transmitir información por la red. Protocolo de comunicaciones estándar que comunica servidores, proxies y clientes. Permite la transferencia de documentos web, sin importar cual es el cliente o cual es el servidor. 1.3. ¿Qué es el DNS? 1.4. ¿Qué es el hosting? 1.5. Navegadores
